# Amazone-Clone

# Overview

This project is a front-end clone of the Amazon homepage built using HTML and CSS. It recreates the basic layout and design of the original website.

# Features

* Responsive navigation bar with logo, search bar, and cart section
* Hero banner section with promotional content
* Product category cards (Health, Clothing, Electronics, etc.)
* Clean and structured layout using CSS
* Basic hover effects for interactive elements

# Technologies Used
* HTML5
* CSS3

# Purpose
 The main goal of this project is to practice and improve front-end development skills by building a real-world UI clone. It demonstrates understanding of layout design, positioning, and styling.

# Project Structure

* index.html – Main HTML file
* style.css – Styling file
* src/ – Contains all images used in the project

# Future Improvements

* Add full responsiveness for mobile and tablet devices
* Integrate JavaScript for dynamic functionality
* Enhance accessibility and performance
